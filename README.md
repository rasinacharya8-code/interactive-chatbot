Neural Converse AI

Project Overview
Neural Converse AI is a modern and highly interactive web based chatbot application. It was designed to simulate a dynamic conversational experience using a rule based engine. The interface features a sleek glassmorphism design with responsive micro animations that create an engaging user experience. 

Key Features
1. Dynamic Conversational Engine
The chatbot analyzes user input and responds contextually based on keyword matching logic. It includes fallback responses to maintain a natural flow of conversation.

2. Interactive Typing Indicator
To simulate a human like interaction, a custom typing indicator appears with a randomized delay before the bot delivers a response.

3. Premium User Interface
The application utilizes a modern dark mode aesthetic with vibrant gradient accents. The glassmorphism effect on the main chat container provides a state of the art visual experience.

4. Micro Animations
Messages smoothly fade and slide into view, and buttons feature subtle hover states to encourage user interaction.

Technologies Used
1. ReactJS (via CDN for lightweight client side rendering)
2. Vanilla JavaScript (ES6+ for conversational logic)
3. HTML5 and CSS3 (Custom styling and animations)

Installation and Setup
1. Clone this repository to your local machine.
2. Open the index.html file in any modern web browser.
3. No build tools or package managers are required to run this project.

Future Enhancements
1. Integrating a live backend API for fully dynamic AI responses.
2. Adding conversation history storage using local browser storage.
3. Expanding the rule based engine for more complex offline interactions.
